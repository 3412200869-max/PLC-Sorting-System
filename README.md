# PLC智能分选系统

基于PLC的智能分拣系统设计与实现，用于自动化工程实践学习。

## 项目介绍

本项目模拟工业生产中的自动分拣设备，
通过PLC实现物料检测、输送、分类和报警控制。

项目目标：

- 学习工业自动化项目开发流程
- 掌握PLC程序设计方法
- 熟悉HMI人机交互设计
- 为后续PLC+Python+AI智能化扩展做准备


## 项目功能规划

### 基础控制

- [ ] 电机启停控制

### 分拣功能

- [ ] 传感器检测

### 智能化扩展
