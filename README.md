# PLC智能分选系统

基于PLC的智能分拣系统设计与实现，用于自动化工程实践学习。

## 项目介绍

本项目模拟工业生产中的自动分拣设备，通过PLC实现物料检测、输送、分类和报警控制。

项目目标：

- 学习工业自动化项目开发流程
- 掌握PLC程序设计方法
- 熟悉HMI人机交互设计
- 为后续PLC+Python+AI智能化扩展做准备

## 当前项目阶段

- 阶段0：开发环境与项目初始化——已完成
- 阶段1：需求分析与平台选型——已完成
- 阶段2：系统详细设计——已完成
- 阶段3：PLC程序实现与PLCSIM仿真——实施准备

项目采用 Siemens S7-1200 + TIA Portal，以 S7-PLCSIM 完成软件仿真；WinCC HMI 在软件和许可条件允许时实施。阶段2设计已经收口，目标Git目录结构和现有文档迁移映射已经定版，当前尚未迁移目录或开始PLC程序编码。
