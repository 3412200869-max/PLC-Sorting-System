# PLC智能分选系统

基于PLC的智能分拣系统设计与实现，用于自动化工程实践学习与技术展示。

## 项目介绍

本项目模拟工业生产中的自动分拣设备，通过PLC实现物料检测、输送、分类、计数和报警控制，并逐步完成PLCSIM与WinCC联合仿真。

项目目标：

- 学习接近真实工业项目的开发流程
- 掌握西门子PLC程序设计与调试方法
- 熟悉HMI人机交互设计
- 为后续PLC、Python与AI扩展预留接口

## 当前PLC变量

| 变量名称 | 地址 | 功能 | 当前状态 |
|---|---|---|---|
| `Start_Button` | `%I0.0` | 启动按钮 | 已使用 |
| `Stop_Button` | `%I0.1` | 停止按钮 | 已使用 |

## 当前注意事项

- 当前实际地址与阶段2《I/O点表V1.0》部分定义存在冲突；新增传感器、急停和执行机构逻辑前，应发布《I/O点表V1.1》并统一地址基线。
- `Motor_Output`目前仅完成变量预留，不代表传送带电机控制模块已经完成。
- 编译通过仅表示程序语法与组态检查通过，启动、保持、停止等行为仍需通过S7-PLCSIM运行测试确认。
